# The Suit - Code Folder

## Overview
This folder contains all software, firmware, and related code for **The Suit** project - the smart compression base layer with sensors, cooling, and future exoskeleton integration.

### Subfolders
- **Firmware/**: C++/Arduino/ESP32 code for microcontrollers, sensor reading, actuator control.
- **Python/**: Scripts for data logging, analysis, backend processing, AI integration.
- **Web/**: HTML, CSS, JavaScript, AJAX for dashboards and web interfaces.
- **Mobile/**: Mobile app code (Flutter, React Native, etc.) if applicable.
- **Libraries/**: Shared or third-party libraries.
- **Tests/**: Unit and integration tests.
- **Docs/**: Documentation, READMEs, diagrams.

## Getting Started
1. Install VS Code or Arduino IDE.
2. For firmware: Use PlatformIO or Arduino IDE.
3. For Python: `pip install pyserial matplotlib pandas` (lightweight for older laptops).

More details in subfolder READMEs.
