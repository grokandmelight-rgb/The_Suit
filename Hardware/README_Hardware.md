# Hardware

This folder contains all physical build documentation for The Suit.

## Subfolders
- BaseLayer: Compression suit, materials, sewing patterns
- Cooling: Pumps, tubing, fluid systems
- Sensors: Placement, wiring, calibration
- Actuators: Motors, pneumatics, compression mechanisms
- Power: Batteries, charging, power distribution

See Project_Overview.md in /Docs for overall specs.
